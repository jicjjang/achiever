# Achiever
Share saved data in your group.

[![Achiever Demo](./static/readme/readme_image.gif)](https://youtu.be/nzAhe7PffbQ)

## Install
~~~bash
$ git clone https://github.com/jicjjang/achiever
$ cd achiever
$ npm install
~~~

## Usage
~~~bash
$ npm run start
~~~

## Test
~~~bash
$ npm run test
~~~
